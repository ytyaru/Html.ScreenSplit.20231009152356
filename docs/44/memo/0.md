## 3

　区切りをHTML要素`div`にする。

```
+---------+--+---------+
|         |  |         |
|         |要|         |
|         |素|         |
|         |  |         |
+---------+--+---------+
```
```
+-------------------+
|                   |
|                   |
+-------------------+
|        要素       |
+-------------------+
|                   |
|                   |
+-------------------+
```

　ただし以下の場合は複雑になるので対象外。

```
+---------+--+---------+
|         |  |         |
|         |  |         |
+---------+  +---------+
|         余白         |
+---------+  +---------+
|         |  |         |
|         |  |         |
+---------+--+---------+
```

　今までとは別クラスとして実装したほうが良さそう。
